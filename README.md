# Customer Behavior Data Analysis

This project focuses on analyzing customer shopping behavior using Python, SQL, and Power BI. The objective of the project is to clean and process raw retail data, perform exploratory data analysis, generate business insights using SQL, and visualize important trends through an interactive Power BI dashboard.

The project simulates a real-world data analytics workflow commonly used in business environments to understand customer purchasing patterns, customer segmentation, and sales trends.

---

## Technologies Used

* Python

  * Pandas
  * NumPy
  * Matplotlib
  * Seaborn
* SQL
* Power BI

---

## Project Workflow

### 1. Data Cleaning & Preprocessing

* Imported and explored the dataset using Python
* Handled missing values and duplicates
* Converted data into analysis-ready format

### 2. Exploratory Data Analysis (EDA)

* Analyzed customer purchasing behavior
* Identified trends and shopping patterns
* Explored customer demographics and spending habits

### 3. SQL Analysis

* Wrote SQL queries to answer business-related questions
* Analyzed customer segments and product performance
* Extracted insights from transactional data

### 4. Dashboard Visualization

* Created an interactive Power BI dashboard
* Visualized sales trends and customer metrics
* Built charts and KPIs for better business understanding

---

## Key Insights

* Identified high-value customer segments
* Analyzed customer loyalty and spending behavior
* Found top-performing product categories
* Visualized overall sales and customer trends
